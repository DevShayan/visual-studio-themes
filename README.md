# Visual Studio default themes

Visual Studio default themes contain one basic theme, dark.

## Supported Languages

* HTML - CSS - JS
* C++
* JAVA - XML
* Python
---

## Applying theme

### To apply theme:
* Ctrl+Shift+P -> Preferences: Color Theme -> dark (Visual Studio)
